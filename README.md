Zepto SQL Data Analysis Project
Project Overview

This project focuses on analyzing Zepto’s e-commerce inventory dataset using SQL.
The goal of the project is to simulate real-world data analyst tasks such as:

1. Data cleaning
2. Exploratory Data Analysis (EDA)
3. Inventory analysis
4. Pricing analysis
5. Business insight generation

The project demonstrates how SQL can be used to extract meaningful insights from raw retail data.

Objectives
1. Analyze product pricing and discounts.
2. Identify out-of-stock products.
3. Find high-value inventory items.
4. Calculate category-wise revenue potential.
5. Perform business-driven SQL analysis.
6. Practice real-world SQL querying techniques.

Tools & Technologies :
SQL, PostgreSQL ,pgAdmin ,CSV Dataset.

Key Insights

1. Some categories provide significantly higher discounts than others.
2. Several expensive products are frequently out of stock.
3. Revenue contribution is concentrated in a few major categories.
4. Price-per-gram analysis helps identify value-for-money products.
